# SAUVEGARDE SESSION CONTEXTE MON_PS

## Instructions
Sauvegarder l'etat actuel de la session pour pouvoir reprendre plus tard.

## Actions requises:

1. **Creer/Mettre a jour CURRENT_TASK.md** dans `/home/Mon_ps/docs/`
   - Decrire la tache en cours
   - Lister les fichiers modifies/crees
   - Noter les problemes rencontres
   - Indiquer les prochaines etapes

2. **Creer fichier session** dans `/home/Mon_ps/docs/sessions/`
   - Format: `YYYY-MM-DD_NN.md` (NN = numero session du jour)
   - Contenu: resume detaille de ce qui a ete fait

3. **Structure fichier session:**
```markdown
# Session [DATE] - [TITRE COURT]

## Contexte
[Ce qui etait demande]

## Realise
- [Liste des accomplissements]

## Fichiers touches
- [path/fichier] - [action: cree/modifie/supprime]

## Problemes resolus
- [Description probleme] -> [Solution]

## En cours / A faire
- [ ] [Tache incomplete]

## Notes techniques
[Details importants pour reprendre]
```

4. **Confirmer** la sauvegarde avec resume
